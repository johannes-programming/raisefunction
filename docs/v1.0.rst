Introduction
------------

This project provides a function that raises errors passed to it.

Installation
------------

To install raisefunction, you can use pip. Open your terminal and run:

.. code-block:: shell

    pip install raisefunction

Features
--------

``raisefunction.raisefunction(exc: BaseException) -> None`` ``raisefunction.raisefunction(exc: BaseException, cause: Optional[BaseException]) -> None``
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

This function binds the raise command.

``raisefunction.test() -> unittest.TextTestRunner``
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

This project can be tested through its test function.

.. code-block:: python

    import raisefunction
    raisefunction.test()
